How to Set Up Your Google Cloud Vision API Key

  1. Go to the Google Cloud Console:
    https://console.cloud.google.com/
  2. Create a new project:
    Click the project dropdown in the top nav, then click New Project. Give it any name (e.g., OCR Tool) and click Create.
  3. Enable the Vision API:
    In the left sidebar, go to APIs & Services → Library. Search for “Vision API.” Click it, then click Enable.
  4. Create an API key:
    Go to APIs & Services → Credentials. Click + Create Credentials and choose API key.
    A long string will appear — this is your API key. Copy it.
  5. Paste the API key into the OCR tool
    Return to the main page and paste your key into the input labeled “Google Vision API key.”
  6. (Optional) Restrict the key:
    On the key management screen, click Restrict key to:
    • Limit usage to the Vision API
    • Add IP address restrictions (if desired)
    • Set usage quotas

Once your key is ready, return to the OCR upload page and run your job.
