Google Cloud API Key Instructions

Follow the instructions below to obtain an API key linked to your Google Cloud billing account.




  1. Bring up Google Cloud Console:
    While logged into your Google account, go to the Google Cloud Console. If you've never used it before, you may see some setup prompts.
  2. Enable billing:
    Go to Billing and enable. You'll need a credit card. That said, you'll only be billed after exceeding 1,000 pages per month, namely at the rate of $1.50 per subsequent 1,000 pages. The first free months on a new billing account come with a $300 credit. That is, it's easy to make substantial use of this service and still pay nothing. It's possible to track your usage, although not very user-friendly (see instructions on Reddit).
  3. Select project:
    Either create a new project (click New Project> in project dropdown in top nav...), or, if this your first time, leave it on the single default (“My First Project”).
  4. Enable Cloud Vision API:
    In the left sidebar, go to APIs & Services → Library. Search for “Cloud Vision API” (it has a blue logo). Click it, then click the blue Enable button.
  5. Create an API key:
    Go to APIs & Services → Credentials. Click + Create Credentials and choose API key.
    A long string will appear — this is your API key. Copy it and store it. Treat it like you would any financial password. With this key, anyone can charge OCR processing (or other services, if you don't restrict the key) to your account.
  6. (Optional) Restrict the key:
    On the API key management screen in the Google Cloud Console, click the three dots to the right of the relevant key, and choose Edit. Click Restrict key to:
    • Limit usage to the Vision API
    • Add IP address restrictions (if desired)
    • Set usage quotas
  7. Use the key:
    You are now ready to return to the OCR page and paste your key into the input box labeled “Google Cloud Vision API key.”
