file upload processing

Upload a file instead of pasting text. The input window handles up to 128mb, but file upload may be more convenient for large texts.

Transliteration: Set input and output schemes, then use the "Transliteration" button under File Upload. Upload your file on the next screen; output goes to your downloads.

Word splitting: Choose settings, then use the "Split" button. The 2024 model is slower than the 2018 one when preserving compound hyphens (some discussion here).

Meter identification: Input must have exactly one verse per line with no extraneous material. See the "Scan GRETIL" page for details. Starter examples (Bhagavadgītā):

  • BhG first ten
  • BhG first hundred
  • BhG all

OCR: Upload a PDF with your Google Vision API key. See the OCR instructions.
