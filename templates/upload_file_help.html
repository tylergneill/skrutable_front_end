file upload help

File upload processes an entire text file with one of Skrutable's actions and returns the result as a download. It's a convenience alternative to copy-pasting text into the workbench. Both modes accept up to 128 MB.

how-to

  1. Switch to Upload File mode using the toggle at the bottom of the sidebar.
  2. Choose an action (Transliterate, Identify Meter, Split Words, or PDF OCR)
  3. For the first three, double-check settings, select your file, and click Upload and Process. The result downloads automatically.
  4. For PDF OCR, setup is different; see separate FAQ.

Relevant settings (input scheme, output scheme, splitter model, etc.) are shown live in the upload view and can be adjusted in the sidebar at any time before submitting. It can be helpful to try settings on a small example in the text workbench first before uploading a large file.

per-action tips

Meter identification: Input must have exactly one verse per line with no extraneous material. See the "Scan GRETIL" page for details. Starter examples (Bhagavadgītā):
